## Data Sources

- **Primary**: [USASpending.gov](https://www.usaspending.gov) contract transaction exports
- **Contract Period**: FY2021 - FY2026

## Files

- `index.html` - Interactive dashboard with vendor analysis
- `Contracts_PrimeTransactions_*.csv` - Raw USASpending contract data
- `BIS_Intelligence_and_Licensing_Contracts.csv` - Filtered BIS contracts
- `Analysis_*.md` - Detailed qualitative analysis for each vendor
